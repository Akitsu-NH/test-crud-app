## アプリ名称

セルフ勤務記録(test-crud-app)

## 概要

- 出勤・退勤ボタンで押下した時間を保存(CU)
- 記録した時間を一覧表示(R)
- 内部で出勤・退勤の状態を保存または更新(U)
- 削除ボタンでレコードを削除(D)
